# smart-room-monitoring
Умный комнатный мониторинг

Обзор проекта
Умная система мониторинга условий в помещении с использованием датчиков влажности, температуры, освещения, со2 и уровня шума
Цель проекта
Решает проблему недостаточного контроля за микроклиматом в жилых и рабочих помещениях, что может негативно сказываться на здоровье, продуктивности и комфорте людей.

Краткое описание функционала
- **Мониторинг в реальном времени**: температура, влажность, качество воздуха, освещенность
- **Исторические данные**: хранение показателей за период

Аналогичные проекты
1. **Home Assistant** - открытая платформа для умного дома
2. **OpenHAB** - кроссплатформенное решение для автоматизации
3. **Domoticz** - легковесная система для мониторинга
4. **ThingsBoard** - IoT-платформа для сбора и визуализации данных

Планируемые пользователи и сценарии
- **Домовладельцы**: контроль микроклимата в квартире/доме
- **Офисные работники**: мониторинг условий в рабочих помещениях
- **Аллергики и астматики**: контроль качества воздуха

# Минимальный функционал (к концу семестра)

##  Обязательные функции
1. **Сбор данных**
   - Подключение к датчикам (реальным или эмулированным)
   - Регулярный опрос и сохранение показаний
   - Поддержка минимум 3 типов параметров (температура, влажность, CO2)

2. **База данных**
   - Хранение исторических данных (минимум 30 дней)
   - Быстрый доступ к последним показаниям
   - Резервное копирование данных

3. **Базовые уведомления**
   - Оповещения о выходе параметров за допустимые пределы
   - Настройка пороговых значений
   - Визуальные индикаторы состояния

# Техническая часть

### Backend
- **Язык**: Python 3.9+
- **Фреймворк**: FastAPI (быстрый и современный)
- **База данных**: SQLite (для разработки) / PostgreSQL (для продакшена)
- **ORM**: SQLAlchemy
- **Асинхронность**: async/await для эффективной работы

### Frontend
- **Фреймворк**: React.js + TypeScript
- **Визуализация**: Chart.js или D3.js
- **Стили**: Tailwind CSS
- **Состояние**: Redux Toolkit

### Дополнительные технологии
- **MQTT**: для связи с IoT-устройствами
- **Docker**: контейнеризация приложения
- **Git**: контроль версий
